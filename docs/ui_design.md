## Color & Typography

### Theme
| Element | Value |
|---------|-------|
| Primary color | Deep indigo (`#1A237E`) |
| Accent color | Amber (`#FFC107`) |
| Background | Dark (`#121212`) â€” easy on eyes in practice |
| Card bg | `#1E1E1E` |
| Text primary | White `#FFFFFF` |
| Text secondary | `#B0B0B0` |

### Typography
| Element | Style |
|---------|-------|
| Countdown number | 120sp, bold, monospace |
| Caption (practice) | 48-96sp (configurable), bold |
| Label list item | 16sp, regular |
| Time display | 20sp, monospace |

**Rationale:** Dark theme by default â€” musicians often practice in dimly lit environments. Large countdown numbers are readable from a distance (music stand / propped phone).
